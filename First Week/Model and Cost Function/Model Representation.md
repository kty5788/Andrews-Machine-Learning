훈련 집합을 통해 이 지도 학습 알고리즘이 어떻게 작동되는지 알아 볼 것이다.

<img width="454" alt="스크린샷 2021-10-16 오후 3 26 48" src="https://user-images.githubusercontent.com/63940620/137576424-6e117f05-7d72-4ea9-a4b6-e49e936da60b.png">

1. Training Set (주택 가격 같은 데이터 셋)
2. Learning Algorithm ( 지도 학습 알고리즘 (Supervised Learning Algorithm)을 통해 결과 값을 도출)
3. h (소문자 h는 2.Learning Algorithm의 결과값이다. 다른 말로 가설(hypothesis)라고도 한다.)

(가설이라고 불리우는 이유는?)
  친구가 팔려는 집의 사이즈(x)를 통해 결과값(y)를 '예측'한 것이기 때문이다.

4. 어떻게 가설 h를 표현해야 하는가?
  가설을 hθ(x) = θ0 + θ1x 라고 표현할 것이다. 줄여서 h(x) 
  
왜 h(x)를 저렇게 표현하는가?
  - 자료와 함수를 통해 예측하는 가장 간단한 방법인 일차함수로 표현한다. 물론 비선형함수 같은 복잡한 것도 있기는 하지만 일차함수 같은 선형유형이 쉽기 떄문이다.
  <img width="414" alt="스크린샷 2021-10-16 오후 3 35 54" src="https://user-images.githubusercontent.com/63940620/137576663-feefd030-499e-49ea-8f35-a6e8eefd09df.png">

  이러한 h(x) 처럼 일차함수로 이루어진 선형을 '선형회귀'(단일변량 선형회귀)라고 한다.
  
  '선형 회귀'의 특징 : 1개의 x 값으로 모든 가격을 예측한다.
